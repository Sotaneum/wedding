# Wedding Invitation

이번에 결혼을 준비하시면서 디지털 청첩장이 필요해지셨나요?
시대가 변화하면서 종이 청첩장을 대량으로 제작하기보다는, 디지털 청첩장을 공유하는 흐름이 자연스러워졌습니다.

종이 청첩장을 구매할 때 디지털 청첩장을 함께 제공하는 경우도 있고, 스드메 업체에서 서비스로 제공하는 경우도 있으며, 별도로 유료 제작하여 사용하는 경우도 있습니다.

다만, 업체에서 무료로 제공하는 청첩장의 경우 디테일이 다소 아쉽거나 PC 환경을 충분히 고려하지 않은 사례도 있어, 아쉬운 마음에 직접 제작하게 되었습니다.

화려하지는 않지만 부족함 없이 사용할 수 있는 디지털 청첩장을 목표로 구성하여 실제로 사용했습니다. 이대로 묻히기에는 아쉽다고 생각해 오픈 소스로 공개합니다.

개발자가 아니더라도 일반 사용자도 쉽게 활용할 수 있도록 가이드를 함께 준비했습니다. 필요하신 분들은 자유롭게 사용하시기 바랍니다.

결혼 축하드립니다 ㅎ

단, 유료 판매하거나 유료 상품에 포함하여 사용하는 경우에는 반드시 사전 협의가 필요하며, 원칙적으로 금지합니다.

## 가이드

### repo fork 하기

- 먼저 [Github에 가입](https://github.com/signup)합니다. 이미 가입되었다면 다음 단계로 넘어갑니다.
  - Github은 구글 계정이나 애플 계정으로 쉽게 가입하거나 간단한 정보로 가입이 가능합니다.
- [fork](https://github.com/Sotaneum/wedding/fork)를 클릭하여 Create fork 버튼을 클릭합니다.
- fork된 repo에 접근하신 다음 화면에 setting.json 파일을 찾습니다.
- 우측에 있는 연필모양으로 편집을 시도합니다.

### 공유하기 template 만들기

- 이 서비스는 카카오의 지도(선택 사항)와 공유하기(선택 사항)를 사용합니다. 만약 지도, 공유하기를 `모두` 사용하지 않을 예정이라면, 생략해도 좋습니다.
- 먼저 [카카오 developers에 가입]합니다.
- [앱](https://developers.kakao.com/console/app)에 접속하여 앱 생성 버튼을 클릭 후 아래 내용을 가이드에 따라 기입합니다.

#### 메시지 템플릿 꾸미기 (선택)

- 공유하기를 하지 않는 경우 생략합니다.
- 앱 생성이 완료되면, 생성한 앱을 선택하고, 좌측의 메뉴에서 제품 설정 > 카카오톡 메시지를 선택합니다.
- 메시지 템플릿 빌더를 선택합니다.
- 메시지 템플릿 추가 버튼을 누르고 원하는 모양을 선택한 후 확인을 누릅니다.
- 템플릿 가이드에 따라 꾸미고 저장합니다.

### 서비스하기

- 무료로 서비스를 할 수 있는 방법은 총 두 가지 입니다. 상황에 맞게 선택해서 사용하세요.

#### Github으로 서비스하기 [무료]

> - 무료이지만, 사진과 모든 정보가 모두 외부에 공개됩니다. (kakao developers key 정보, 이미지 다운로드 등)
> - node/npm 등 부가적인 요소를 다운로드하고 설치해야합니다.

- 컴퓨터에 [node](https://nodejs.org/ko/download)를 설치합니다.
- fork했던 repo에 진입하고 상단의 `<>Code` 버튼을 클릭하고 중앙에 있는 링크를 복사합니다.
- 터미널을 켜고 빈 폴더에 진입한 다음 git clone {링크복붙} 명령어를 수행하여 repo를 복사합니다.
- {작성 중...}

#### Vercel으로 서비스하기 [무료]

> - 무료이지만, 일정량이 증가하면 사이트가 중단될 수 있습니다. (사진의 개수에 따라 다르지만 하루에 200명정도는 견딜 수 있음)
> - vercel 이라는 서비스를 새롭게 가입해야합니다. (무료)

- [vercel에 가입](https://vercel.com/signup)합니다.
- hoddy를 선택하고 이름을 기입 후 가입을 진행합니다.
- 우측에 있는 Add New 버튼을 클릭하고 Project를 선택합니다.
- 하단의 Import Git Repository에서 Add Github Account를 선택합니다. 자동으로 Github 연동 화면이 표시되며, 권한을 수락되면서 목록이 표시됩니다.
- Wedding repo의 import 버튼을 클릭합니다.
- Project Name을 원하는 이름을 지정 후 deploy 버튼을 클릭합니다.
- Congratulations가 뜨면 Continue to Dashboard 버튼을 클릭합니다.
- 좌측의 Domains를 선택하고 Add Existing 버튼을 클릭합니다.
- 돋보기 옆에 {원하는문자열}.vercel.app 형태로 기입하고 save 버튼을 누릅니다. 이 url 여러분의 모바일 청접장 링크가 됩니다.

### 카카오 developers 도메인 등록하기 (선택)

- 공유하기나 지도를 모두 사용하지 않는 경우 생략해도 됩니다.
- 카카오의 경우 지정한 url만 동작하도록 되어있어서 생성한 url를 넣어두어야합니다.
- developers에 접속하여 app 에 들어간 다음 좌측의 앱 > 제품 링크 관리에 들어갑니다.
- 웹 도메인 수정 버튼을 클릭하고 전 단계에서 추가했던 {원하는문자열}.vercel.app 를 넣고 저장합니다.
